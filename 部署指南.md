# ChatWing 部署指南 - 从零到手机安装

本文档详细描述从一台刚装好的 Windows 电脑到手机上出现 ChatWing APK 的完整流程。
预计耗时：首次配置 40-60 分钟，后续构建仅需 2 分钟。

---

## 第一章：安装必要工具

### 1.1 安装 Flutter SDK

1. 打开浏览器，访问 https://docs.flutter.dev/get-started/install/windows
2. 点击下载链接 `flutter_windows_3.24.0-stable.zip`（约 1GB）
3. 下载完成后，将 zip 文件解压到 `D:\` 根目录
   - 右键 zip → 全部提取
   - 目标路径填 `D:\`
   - 等待解压完成
4. 确认解压后目录结构：`D:\flutter\bin\flutter.bat` 存在

### 1.2 配置环境变量

1. 按 `Win + R`，输入 `sysdm.cpl`，回车
2. 点"高级"标签 → 右下角"环境变量"
3. 在"系统变量"区域点"新建"：
   - 变量名：`FLUTTER_HOME`
   - 变量值：`D:\flutter`
4. 找到 `Path` 变量 → 双击 → 点"新建" → 添加：
   - `D:\flutter\bin`
5. 一路点"确定"关闭所有窗口

### 1.3 验证 Flutter 安装

1. 打开"命令提示符"（`Win + R` → 输入 `cmd` → 回车）
2. 输入命令：
   ```
   flutter --version
   ```
   如果看到 Flutter 3.24.0 等版本信息，说明安装成功。

### 1.4 安装 Git

1. 访问 https://git-scm.com/download/win
2. 下载安装程序，双击运行
3. 安装步骤全部保持默认选项即可
4. 安装完成后验证：`git --version`

## 第二章：在本地构建 APK

### 2.1 打开项目目录

```bash
D:
cd D:\work++\ChatWing
```

### 2.2 安装项目依赖

```bash
flutter pub get
```

等待命令执行完成。这会自动下载项目所需的第三方库：
- `provider` - 状态管理
- `sqflite` - 本地数据库
- `http` - 网络请求
- `flutter_secure_storage` - 加密存储

### 2.3 接受 Android SDK 许可

```bash
flutter doctor --android-licenses
```

一直按 `y` + 回车 接受所有许可协议（约 7 次）

### 2.4 构建 APK

```bash
flutter build apk --release
```

构建耗时约 2-5 分钟。看到以下信息即成功：

```
✓ Built build\app\outputs\flutter-apk\app-release.apk (XX.X MB)
```

### 2.5 常见错误对照表

| 错误 | 原因 | 解决 |
|------|------|------|
| `Android SDK not found` | 没装 Android SDK | 装 Android Studio，启动后自动安装 |
| `JAVA_HOME not set` | 没装 JDK | `flutter doctor` 会提示安装 |
| `Could not find aapt2` | SDK 组件不全 | `sdkmanager "build-tools;34.0.0"` |
| `Unable to find git` | 没装 Git | 装 Git for Windows |
| Gradle 配置失败 | 网络问题 | 检查能不能访问 maven.google.com |

### 2.6 安装到手机

**方法一：数据线**
```bash
# 复制到桌面
copy build\app\outputs\flutter-apk\app-release.apk %USERPROFILE%\Desktop\
# 手机弹出"允许 USB 调试" → 确定后传文件即可
```

**方法二：微信**
打开微信电脑版 → 文件传输助手 → 拖拽 APK 文件到对话框 → 手机上下载安装

**方法三：网盘**
上传到百度云/阿里云盘 → 手机下载安装

## 第三章：用 GitHub Actions 自动构建（推荐）

### 3.1 注册 GitHub

1. 访问 https://github.com/signup
2. 输入邮箱 → 设置密码 → 验证邮箱
3. 选择 Free 套餐

### 3.2 创建仓库

1. 登录 GitHub，点右上角 `+` → `New repository`
2. 仓库名填 `ChatWing`
3. **不要勾选** "Add a README"、"Add .gitignore"、"Choose a license"
4. 点 `Create repository`
5. 创建后页面会显示命令列表，保留此页面不关

### 3.3 推送代码

打开命令提示符并执行：

```bash
D:
cd D:\work++\ChatWing

git branch -M main
git remote add origin https://github.com/你的用户名/ChatWing.git
git push -u origin main
```

### 3.4 创建 Personal Access Token

如果 push 时提示输入密码：

1. 访问 https://github.com/settings/tokens
2. 点 `Generate new token (classic)`
3. Note 填 `ChatWing`，过期时间选 `No expiration`
4. 勾选 `repo` → 点底部 `Generate token`
5. **立即复制** token（`ghp_` 开头，只显示一次）
6. 回到弹窗，粘贴 token 作为密码

### 3.5 查看构建进度

1. 打开 https://github.com/你的用户名/ChatWing
2. 点顶部 `Actions` 标签
3. 黄色圆点 = 构建中，绿色对勾 = 完成（约 5-8 分钟）

### 3.6 下载 APK

1. 点绿色对勾 → 页面底部 `Artifacts`
2. 下载 `chatwing-release-apk.zip`
3. 解压得 `app-release.apk`

### 3.7 后续更新

每次改完代码后：

```bash
cd D:\work++\ChatWing
git add .
git commit -m "改了啥"
git push
```

推送后自动触发 Actions 构建，5 分钟后去下载最新 APK。

## 第四章：手机上首次配置

1. 安装后打开 ChatWing，允许悬浮窗和通知权限
2. 底部导航栏点"设置"
3. 输入 DeepSeek API Key（https://platform.deepseek.com 获取，新用户送 500 万 token）
4. 保存 API 配置
5. 点"人物小传"填写你的个人信息（年龄、星座、MBTI、口癖等）
6. 返回设置页，启动所需服务
